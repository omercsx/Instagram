# Instagram Clone App

Full-stack Instagram Clone with React Native and AWS.

## Used Technologies

- React Native CLI with Typescript
- AWS Amplify AWS Cognito
- GraphQL
- Amazon S3
- Firebase Cloud Messaging
- and more...
